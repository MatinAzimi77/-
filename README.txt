راهنمای اجرای سریع وب‌اپلیکیشن سیستم مدیریت صنعتی:

۱. ابتدا مطمئن شوید پایتون روی سیستم شما نصب است.
۲. این فایل زیپ را در یک پوشه جدید استخراج (Extract) کنید.
۳. نرم‌افزار Command Prompt (یا cmd) ویندوز را باز کنید و با دستور cd به مسیری که پوشه را استخراج کردید بروید.
۴. دستور زیر را برای نصب کتابخانه‌های مورد نیاز وارد کنید:
   pip install -r requirements.txt
۵. پس از اتمام نصب، برای روشن کردن سرور اپلیکیشن دستور زیر را وارد کنید:
   streamlit run app.py

مرورگر شما به طور خودکار باز شده و پنل مدیریتی آماده استفاده خواهد بود!
